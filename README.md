# kakeibo
※このアプリは開発途中です。
現金の入出金をざっくりと記録するためのアプリです。

## アプリについて
Googleアカウントでログインし、ユーザーごとに入出金の記録をつけることが出来ます。
よく使う品目は登録することも可能です。

## 実行方法

```
$ git clone https://github.com/kaboko14/kakeibo.git
$ npm install
$ npm run serve
```

## 使用言語・環境
Vue.js + Firestore + FirebaseAuthentication
